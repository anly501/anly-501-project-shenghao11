Perform K-means

The purpose of k-means clustering is to divide n observations into k clusters, each belonging to the cluster with the closest mean, as the prototype of that cluster. I am using elbow method to find the optimal number of the clusters.

Code 
# for k means clustering we will use the elbow method to find the optimal number of clusters.
distortions=[]
inertias=[]
k=11
OMP_NUM_THREADS=2
for k in range(1,k):
    kmeanModel = KMeans(n_clusters=k,init = 'k-means++',random_state=42)
    kmeanModel.fit(x)

    distortions.append(sum(np.min(cdist(x,kmeanModel.cluster_centers_,'euclidean'),axis=1))/x.shape[0])

    inertias.append(kmeanModel.inertia_)
    evaluation=pd.DataFrame.from_records({'Cluster':np.arange(1,k+1),'Distortion':distortions,'Inertia':inertias})
